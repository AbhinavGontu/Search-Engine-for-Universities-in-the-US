Tips and Tricks

Display Oddities

Noticing Canvas homepage with flickering course cards, or the assignments page not displaying correctly? Make sure your browser is up-to-date. You can also try using a different browser.

Notes Column in the Grade Book

Try using the "note" column in the Canvas grade book to keep small bits of information on your students. You have to make the notes column visible before you can use it.

Voice over PowerPoint

If you are using Kaltura Capture to record narration over a PowerPoint slide you might consider using PowerPoints “Narration” feature (rather than Kaltura Capture).
