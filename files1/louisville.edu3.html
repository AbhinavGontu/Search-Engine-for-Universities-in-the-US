System Announcements

  • NEW: Respondus now works with Chromebooks with some limitations (Beta) (Thursday, February 18, 2021)

    More information Chromebook used with LockDown Browser and Respondus Monitor. This is an option in the Respondus Lockdown dashboard for exam(s). 20210218

  • NEW: Respondus Lockdown Browser + Instructor Live Proctoring (Beta) (Thursday, February 18, 2021)

    Respondus LockDown Browser has added a new feature, Instructor Live Proctoring, which allows it to be used in conjunction with video conferencing systems such as Microsoft Teams. This synchronous testing option allows instructors to monitor the physical testing environment through a video conferencing system while LockDown Browser helps to secure the virtual testing environment. To learn more about this feature, including its strengths and limitations, please review the LockDown Browser + Instructor Live Proctoring Information Module. This is an option in the Respondus Lockdown dashboard for exam(s). 20210218

  • ATTENTION ALL: LOCKDOWN BROWSER UPDATE REQUIRED FOR WINDOWS VERSION (Tuesday, February 9, 2021)

    A hotfix update to LockDown Browser for Windows has been released: version 2.0.7.00. This release contains an update to the underlying Chromium engine, plus a fix for a recent Blackboard Learn SaaS update. Existing installs of LockDown Browser will NOT be auto-updated at this time. To obtain the latest version (2.0.7.00) use the "Check for Update" feature or run the full installation program. 20210209

  • ATTENTION: ALL: Lockdown Browser PDF issue with Macs (Monday, February 1, 2021)

    Currently Lockdown Browser (2.0.7) tests containing attached PDFs, create an issue with students using Macs. Students will get a message, "link is blocked" for the Mac version while in the test. Respondus has indicated that this issue will be resolved in their upcoming update, 2.0.7.01. 20210121
