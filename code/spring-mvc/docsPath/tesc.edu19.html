About the Newspaper Collection at the New Jersey State Library

Welcome to the New Jersey State Library Newspaper Research Guide.

♦ Newspapers and microfilm are located on the 4th floor of the library.

♦ The NJ State Library currently receives about 18 NJ newspapers representing a wide range of geographic regions throughout the state. We also receive 4 national papers.

♦ About 70 older newspapers are available on microfilm.

♦ Find NJ State Library Newspapers Organized by County.

♦ Find a subject index to the Newspaper Clipping File covering the 1950's to the 1980's for several major NJ papers. More information about the clipping file can be found in the Historic and Current Newspapers section.

► Please note: there are some missing newspaper issues/microfilm reels in the collection. Please call ahead to verify that a newspaper or microfilm reel you need is available before traveling to the library.

  • Newspapers at NJSL - Alphabetical List
    Complete list of newspapers in print and on microfilm held at the New Jersey State Library.

The New Jersey State Archives, located nearby at 225 West State Street, Trenton NJ, also has a large collection of historical newspapers on microfilm.
