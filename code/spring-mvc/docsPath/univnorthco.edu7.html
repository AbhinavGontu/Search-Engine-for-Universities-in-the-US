Keyboard Shortcuts

Esc

Closes this screen, menu, card, street view, fullscreen photos, and gallery.

When inside of a gallery, ⇆ to go to the next image.
When search options are shown ⇅ can highlight an item. Press Enter to select the item.

+ and -

Zoom in and out when focused on the map.

f

Sets the focus on the map.

k

Opens this window

s

Set focus to search.

m

Toggle the menu, sets focus on the first item. Use Tab and Shift+Tab to navigate through items. Press Enter to toggle layer.
