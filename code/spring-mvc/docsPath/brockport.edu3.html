NetID Lookup and Password Change System


What is a NetID?

Your NetID is a username used to access Brockport hosted services. All students, faculty, and staff have a NetID. If you have forgotten your NetID password, it can be reset using the link below.

I don't know my current password

If you know your current password, it can be changed using the NetID Information and Password Reset page from within Web Banner. You can also use that page to set passwords for other accounts you may own, such as a Brockport generic account.
