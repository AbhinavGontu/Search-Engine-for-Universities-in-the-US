Learn how we can help you advance your career, explore new opportunities and change your life.

Are you a graduate, transfer or first-time undergraduate? Please select an option to see a list of upcoming events.

If you are an international student, veteran student, or are interested in an event focused on a specific degree program, please select the student option below that best describes you for a full list of upcoming events.

Note: All events are hosted on Zoom. Prior to joining the event, please ensure you have installed the Zoom application and/or have updated to the latest version. Some features during the presentation may not be available if your version is outdated.
