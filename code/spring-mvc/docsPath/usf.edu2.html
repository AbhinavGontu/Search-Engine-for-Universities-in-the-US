The USF Directory contains contact information for all current faculty and staff. Each directory listing will include the individual's name, title, department, campus, email address, phone number and campus mail point. You can search for individuals by name or by email address, or click on Advanced Search to filter the results by campus, college or department.

Information on updating your directory listing can be found on the Human Resources website.
