Safe Browsing

Logging in using a shared computer?

"Shibboleth" is a technology that allows access to many resources with a single login. While this is helpful on personal systems, this method poses a risk for users of shared computers where stored login credentials may be inappropriately used. To protect from this risk, when using a shared computer you should ensure that "Private Browsing" is turned on within the web browser. In addition, from a shared computer you should always close ALL browser windows and tabs when you are finished.

Instructions for configuring Private Browsing:

Internet Explorer - Click the Tools menu and select InPrivate Browsing.

Firefox on Windows - Click the Firefox or Tools menu and select Start Private Browsing.

Firefox on Mac - Click the Tools menu and select Start Private Browsing.

Safari - Click the Safari menu and select Private Browsing.

If you have any questions, contact your university Help Desk.
