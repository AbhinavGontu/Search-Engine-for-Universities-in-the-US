Course Offerings

The Course Search screen, available through STARS, allows you to search the current class schedule or projected schedule for specific course criteria, open sections, and department courses.


Course Search

     
  • Launch STARS
    Current students may login and interactively use Course Search while registering for classes. STARS is available from 3 AM - Midnight (Central time).
  • STARS Help
    STARS Help is available from 8 AM - 4:30 PM (Central time).
  • Archived Offerings
    Previous course offerings are available in PDF format.
  • Registration Information

Western Illinois University does not publish a printed version of the class schedule. This website provides the official class schedule. The Office of the Registrar reserves the right to change procedures listed on this website, if necessary. Class days, times, and location subject to change. To help your fellow student, do not register for a class you do not plan to attend.
