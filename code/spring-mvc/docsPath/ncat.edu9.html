Laptop Loan Program

In response to the COVID-19 pandemic, F. D. Bluford Library fully supports distance and virtual learning initiatives. The Library has created a Laptop Lending Program that offers enrolled students the availability to checkout laptops for academic support and success. These devices can be borrowed on a first-come, first-served basis. Students outside of Guilford County will receive top priority.
